# 💰 Expense Tracker

A simple and interactive Expense Tracker built using HTML, CSS, and JavaScript. This project helps users track their daily expenses and manage their budget efficiently.

## 🚀 Features

* Add income and expenses
* View balance updates in real-time
* Track transaction history
* Delete transactions
* Simple and user-friendly interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
